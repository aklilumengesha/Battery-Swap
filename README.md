# BatterySwap

Battery swapping platform for electric vehicles.

## Project Structure

```
BatterySwap/
├── frontend/     - Next.js TypeScript frontend
└── backend/      - Django REST API backend
```

## Frontend

Built with Next.js, TypeScript, Redux Toolkit, and Ant Design.

See `frontend/README.md` for setup instructions.

## Backend

Built with Django, Django REST Framework, and PostgreSQL.

See `backend/README.md` for setup instructions.
